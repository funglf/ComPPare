\doxysubsubsection*{Usage}

This file supports two usage patterns for formatting console output with ANSI escape codes\+:

\doxysubsubsubsection*{1. Persistent toggles}


\begin{DoxyItemize}
\item Applied with stream insertion ({\ttfamily \texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<}}).
\item Remain active until explicitly turned off with the corresponding {\ttfamily \+\_\+\+OFF} code.
\item Examples\+: {\ttfamily BOLD} / {\ttfamily BOLD\+\_\+\+OFF}, {\ttfamily RED\+\_\+} / {\ttfamily RED\+\_\+\+OFF}.
\end{DoxyItemize}

\#define ANSI\+\_\+\+DEFINE(\begin{DoxyParamCaption}\item[{}]{NAME}{, }\item[{}]{ON}{, }\item[{}]{OFF}{}\end{DoxyParamCaption})}



Macro to define ANSI escape codes for text styling and colors. 

This macro generates a {\itshape pair of classes} and its {\itshape instances} to represent an ANSI style or color. The resulting API supports both\+:
\begin{DoxyItemize}
\item {\bfseries{Persistent toggles}} (turn style/color on or off in the stream)
\item {\bfseries{Scoped wrappers}} (apply ON/\+OFF just around a single value)
\end{DoxyItemize}
