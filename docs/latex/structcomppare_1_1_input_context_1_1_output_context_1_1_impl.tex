struct comppare\+::\+Input\+Context$<$ Inputs $>$\+::\+Output\+Context$<$ Specs $>$\+::\+Impl}
Internal container representing one registered implementation. 

Each {\ttfamily \doxylink{structcomppare_1_1_input_context_1_1_output_context_1_1_impl}{Impl}} bundles together\+:
\begin{DoxyItemize}
\item the {\itshape user function} ({\ttfamily fn}) under a given name,
\item a pointer to the input tuple,
\item a back-\/reference to the owning {\ttfamily \doxylink{classcomppare_1_1_input_context_1_1_output_context}{Output\+Context}},
\item and optionally, plugin-\/managed output storage.
\end{DoxyItemize}

This allows the framework to keep track of multiple competing implementations of the same operation (e.\+g. reference vs optimized), and to attach correctness/performance plugins such as Google Benchmark or NVBench.
