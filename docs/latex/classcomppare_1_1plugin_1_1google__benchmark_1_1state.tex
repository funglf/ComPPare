\doxysubsection{Detailed Description}
State class to manage the benchmark state. 

This class is a singleton to ensure a single instance throughout the program. It provides methods to set and get the benchmark state {\ttfamily benchmark\+::\+State \&}. The state is used to interact with the Google Benchmark library.
