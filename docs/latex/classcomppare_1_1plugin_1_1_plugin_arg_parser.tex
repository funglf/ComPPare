\doxysubsection{Detailed Description}
Argument parser for plugin command-\/line arguments. 

This class is responsible for parsing command-\/line arguments specific to a plugin. Each plugin can have its own set of arguments, which are prefixed by a unique header. The parser supports both {\ttfamily -\/-\/header=value} and {\ttfamily -\/-\/header value} formats for specifying arguments.

Example usage\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{classcomppare_1_1plugin_1_1_plugin_arg_parser_a070fe4a706fc8b608f6868c6e47f8428}{PluginArgParser}}\ parser(\textcolor{stringliteral}{"{}-\/-\/myplugin"{}});}

\end{DoxyCode}


In Command Line\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{-\/-\/myplugin=\textcolor{stringliteral}{"{}-\/-\/flag=value"{}}}
\DoxyCodeLine{-\/-\/myplugin\ \textcolor{stringliteral}{"{}-\/-\/flag1\ value1\ -\/-\/flag2=value2"{}}}

\end{DoxyCode}
 Then {\ttfamily parser.\+parse(argc, argv)} will extract the quoted values as arguments for the plugin.
