Applying ANSI styles/colors to a value in a scope. 

The {\ttfamily \doxylink{classcomppare_1_1internal_1_1ansi_1_1_scoped_ansi_wrapper}{Scoped\+Ansi\+Wrapper}} class applies an ANSI "{}on"{} code before the value and an "{}off"{} code immediately after. This makes styles and colors {\itshape scoped} to the wrapped value, unlike persistent toggles.

Example\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{std::cout\ <<\ comppare::internal::ansi::BOLD(\textcolor{stringliteral}{"{}Bold\ Text"{}})}
\DoxyCodeLine{\ \ \ \ \ \ \ \ \ \ <<\ \textcolor{stringliteral}{"{}\ normal\ text"{}};}

\end{DoxyCode}
 The text {\ttfamily "{}\+Bold Text"{}} is printed in bold, and the style is automatically reset afterwards.

{\em T} & The type of the value to be wrapped. Must satisfy {\ttfamily Streamable} (i.\+e. it can be inserted into a {\ttfamily std\+::ostream}).\\

\end{DoxyTemplParams}
